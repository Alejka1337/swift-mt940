# revolut-to-mt940

Конвертер из CSV-выписки Revolut в формат **MT940** (SWIFT) — пригодно для бухгалтерских систем.

**Язык:** Python  
**Управление зависимостями:** Poetry

---

## Что делает проект

Принимает CSV-файл выписки из Revolut, парсит операции и генерирует файл `mt940.txt` в формате MT940 (используется бухгалтерами/банками для загрузки выписок).

---

## Особенности

- CLI-интерфейс: `revolut2mt940 convert input.csv -o mt940.txt`
- Поддержка основных типов операций (debit/credit), валют и простая валидация.
- Лёгкая настройка шаблона счёта и метаданных (параметры через ENV или файл конфигурации).
